Go Networking Flow: Server & Client

Go Server

TCP: Reliable connection for login/chat

ln, _ := net.Listen("tcp", ":7777")
for {
    conn, _ := ln.Accept()
    go handleConnection(conn)
}

UDP: Fast updates for positions/movement

addr, _ := net.ResolveUDPAddr("udp", ":7777")
conn, _ := net.ListenUDP("udp", addr)
buf := make([]byte, 1024)
for {
    n, clientAddr, _ := conn.ReadFromUDP(buf)
}
⇄

C# Client

TCP: Connect to server

TcpClient client = new TcpClient("localhost", 7777);
NetworkStream stream = client.GetStream();
byte[] message = Encoding.ASCII.GetBytes("Hello Server");
stream.Write(message, 0, message.Length);

UDP: Send movement updates

UdpClient udp = new UdpClient();
IPEndPoint serverEP = new IPEndPoint(IPAddress.Loopback, 7777);
byte[] data = Encoding.ASCII.GetBytes("PlayerMove:x=10,y=20");
udp.Send(data, data.Length, serverEP);

Flow Explanation: TCP handles reliable messages like login or chat, while UDP handles real-time updates like positions. The arrows show bidirectional communication between Go server and C# client.
