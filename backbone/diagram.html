Lagtopia System Overview

Backbone 😀

The all-seeing router and brain of the system. Manages live TCP connections, tracks clients, and handles tokens. Sends events to the dashboard so you can see everything happening in real time.
  • Tracks connected clients
  • Manages token issuance and verification
  • Streams events to Backpanel
  • Filters or throttles info to avoid chaos

Backpanel 👁️

Your hacker-cool dashboard, built with a TUI. Displays live client connections, tokens, and logs. Can also take CLI-style commands for direct interaction with Backbone.
  • Shows connected clients
  • Displays token status and stats
  • Logs events and alerts
  • CLI input for commands like kick or regenerate tokens

Spine 💀

The SQL persistence layer. Stores token states, logs, and history, so nothing is lost when the system restarts. Allows historical queries and analytics.
  • Records token and client history
  • Stores event logs
  • Enables historical queries and analytics
